Python Accessible Studies

Overview:
This repository contains an accessibility-first, beginner-friendly Python study plan, exercises, quizzes, starter code, tests, and setup instructions for Windows 11 with JAWS and iPhone access.

Contents:
- README.txt (this file)
- study_plan.csv and study_plan.txt (12-week schedule)
- quizzes/ (text and CSV quizzes with answers)
- exercises/ (text prompts)
- starter_code/ (.py example scripts)
- tests/ (basic unittest files)
- requirements.txt (requests, beautifulsoup4, pytest)
- .vscode/settings.json (accessibility settings)
- install_and_setup_windows.txt (step-by-step commands for Windows + JAWS)
- iphone_access.txt (how to access via iPhone)
- project-templates/ (gitignore and template README)

Accessibility notes:
- All content is provided as plain text, CSV, or simple Python scripts to ensure screen-reader compatibility.
- Prefer the plain-text files and CSV files when using a screen reader.
- Use VS Code with "editor.accessibilitySupport" set to "on" and the integrated terminal for coding. Use JAWS with VS Code; see install_and_setup_windows.txt for tips.

How to use:
- Clone the repo to your Windows PC or access via the GitHub app on iPhone.
- Follow install_and_setup_windows.txt to create a Python venv and install dependencies.
- Work through study_plan.txt or study_plan.csv weekly, complete exercises/ files, and run starter_code scripts in the integrated terminal.
